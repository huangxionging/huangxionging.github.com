// 用于判断字符串是否等于 <null>
#define IS_EQUAL_TO_NULL(string) ([string isEqualToString: @"<null>"] || [string isEqualToString: @“(null)”] || [string isEqualToString: @“"])

// 用于将其类型的数据转化成 NSString 类型
#define TO_STRING(id) [NSString stringWithFormat: "%@", id]

从此以后 MVC 中的 M 模型中的字符串 NSString 和 NSNumber 都改成 NSString 用 TO_STRING来转换

假如 TO_STRING(id) 中 id 是 nil, 得到的结果是 @“(null)”, id 是 NSNull 类型 结果会是 @"<null>”
这确实需要经验才能知道, 庆幸自己发现了这些问题
