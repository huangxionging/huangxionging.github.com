警告框是由UIAlertView创建的，用于给用户警告或者提示，最多有两个按钮，超过两个就应该使用操作表。由于在iOS中，警告框是模态的，意思是不关闭它就不能做别的事情，一般情况下，警告框使用的场景有如下几个。
1）应用不能继续运行，例如，无法获得网络数据功能不能完成的时候，给用户一个警告，这种警告框只需要一个按钮。
2）询问另外的解决方案。好多应用在不能继续运行时，会给出另外的解决方案，让用户去选择。例如网络无法连接时，询问是否可以使用3G网络
3）询问对操作的授权。当应用访问用户的一些隐私信息时，需要用户授权。

如果想给用户多种选择，比如想把应用发给Facebook或者新浪微博或者微信等平台，就应该使用操作表。

- (IBAction)testAlertView:(id)sender
{
    UIAlertView *alertView = [[UIAlertView alloc] initWithTitle: @"Alert" message: @"Alert text goes here" delegate: self cancelButtonTitle:@"No" otherButtonTitles: @"Yes", nil];
    [alertView show];
}

#pragma mark-- 实现UIAlertViewDelegate
- (void) alertView:(UIAlertView *)alertView clickedButtonAtIndex:(NSInteger)buttonIndex
{
    if (buttonIndex == 0)
    {
        [self testAlertView: self];
    }
    NSLog(@"buttonIndex = %d", buttonIndex);
}

- (IBAction)testActionSheet:(id)sender
{
    UIActionSheet *actionSheet = [[UIActionSheet alloc] initWithTitle:@"分享给：" delegate:self cancelButtonTitle:@"取消" destructiveButtonTitle: nil otherButtonTitles:@"Facebook", @"新浪微博", @"微信", nil];
    actionSheet.actionSheetStyle = UIActionSheetStyleBlackTranslucent;
    [actionSheet showInView: self.view];
}

#pragma mark-- 实现UIActionSheetDelegate
- (void) actionSheet:(UIActionSheet *)actionSheet clickedButtonAtIndex:(NSInteger)buttonIndex
{
    if (buttonIndex == 1)
    {
        [self testActionSheet: self];
    }
}
