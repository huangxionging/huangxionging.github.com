系统默认会有返回按钮，但是英文的back，如果我们想要改成中文，应该在导航控制器在子视图出现之前更改返回按钮的标题。因为 self.navigationItem.backBarButtonItem 设置之后只能在下一个页面使用；所以必须在当前页面设置.

UIBarButtonitem *back = [[UIBarButtonitem alloc] init];
back.title = @"返回";
self.navigationItem.backBarButtonItem = back;
之后的页面就可以看到中文的返回按钮了。


如果自定义的话，应当self.navigationItem.leftBarButtonItem 设置为返回按钮，向左的箭头需要自己切图片；

 UIButton *button = [UIButton buttonWithType: UIButtonTypeCustom];
    button.frame = CGRectMake(0, 0, 44, 44);
    [button addTarget: self action: @selector(back:) forControlEvents: UIControlEventTouchUpInside];
    [button setImage: [UIImage imageNamed: @"back_normal"] forState: UIControlStateNormal];
    button.titleLabel.font = [UIFont systemFontOfSize: 14.0];
    button.imageEdgeInsets = UIEdgeInsetsMake(0, -20, 0, 0);
    button.titleEdgeInsets = UIEdgeInsetsMake(0, -20, 0, 0);
    [button setTitleColor: [UIColor blackColor] forState: UIControlStateNormal];

    button.showsTouchWhenHighlighted = YES;
    [button setTitle: @"返回" forState: UIControlStateNormal];
    // 使用自定义视图来初始化
    UIBarButtonItem *back = [[UIBarButtonItem alloc] initWithCustomView: button];

    // 自定义返回
    self.navigationItem.leftBarButtonItem = back;
