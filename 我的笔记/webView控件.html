网页视图控件webView，是用来加载网页使用的，利用这个控件可以很方便的开发一个浏览器。Redmann‘s browser.zip
这个工程就是我自己利用之前所学的知识所开发的一个简单的浏览器应用。

     首先利用UITextField控件接收用户输入的网址，然后利用-(BOOL) textFieldShouldReturn:(UITextField *)textField，来关闭键盘，并调用Go方法，而Go方法又是点击Go按钮所触发的动作方法，所以在文本控件输入网址时，点击键盘上的返回按钮不仅可以关闭键盘还能调用Go方法来请求网页并加载网页。而界面上的Back按钮与webView的goBack动作连接起来，但单击Back按钮时，发出按钮的touch down事件，这时候会触发webView的goBack动作，webView因此会回退网页，相当于我们平常使用浏览器的后退动作。


Go 方法的内容：

首先获取到UITextField控件的文本字符串，就是获得网址；
然后将字符串转换成NSURL 类型的地址；
再利用这个地址来发送网页请求；
由[NSRequest requiredWithURL: pagePath]来发送网页请求，得到的结果被webView加载到控件上，这样就在网页上显示出网页。
 // 获取文本控件webPath的字符串，也就是获得网址
    NSString *path = self.webPath.text;
    // 将字符串网址转换成地址类
    NSURL *pagePath = [NSURL URLWithString: path];
    // 通过地址类来发送网页请求
    NSURLRequest *request = [NSURLRequest requestWithURL: pagePath];
    // 网页视图控件加载请求的网页
    [self.webView loadRequest: request];
}
